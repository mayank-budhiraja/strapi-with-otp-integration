# SettingsPageTitle

This component is used in order to set the title head tag of a setting page.

## Usage

```
import React from 'react'
import { SettingsPageTitle } from '@strapi/helper-plugin';


const HomePage = () => {
  return (
    <>
      <SettingsPageTitle name="Media Library" />
    </>
  );
};
```
