# PageSizeURLQuery

This component is used to change the page size.

## Imports

```js
import { PageSizeURLQuery } from '@strapi/helper-plugin';
```

## Usage

<Canvas>
  <Story name="base">
    {() => {
      const [{ query }, setQuery] = useQueryParams();
      useEffect(() => {
        setQuery({ pageSize: 10 });
      }, []);
      return (
        <Main>
          <Box paddingTop={6}>
            <PageSizeURLQuery />
          </Box>
        </Main>
      );
    }}
  </Story>
